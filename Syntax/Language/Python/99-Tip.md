## 코딩

- for 반복문을 쓸 때, 반복자가 아래 쓰이지 않는다면 _로 대체할 수 있음

```python
n = 5
m = 9

star = '*'

for _ in range(m):
    for _ in range(n):
        print(star, end = '')
    else:
        print()
```
