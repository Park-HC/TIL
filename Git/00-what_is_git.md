# Git

## Git이 무엇일까?

### 분산 버전 관리 프로그램

* 버전: 컴퓨터 소프트웨어의 특정 상태
* 버전 관리 프로그램: 컴퓨터 소프트웨어의 특정 상태(시점)들을 관리하는 프로그램
* 분산 버전 관리: 중앙집중식 버전 관리와 달리, 버전의 기록이 분산되어 저장되어 있어서 정보 소실 등 보안 위협에 보다 안전하다.
